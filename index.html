download

Grabs the repo with all 9 sounds included.

git clone https://github.com/deepika-builds/meme-terminal.git
2
install

Copies sounds to your home folder and hooks into your shell.

cd meme-terminal && ./install.sh
3
restart terminal

Close terminal (Cmd+Q) and reopen. Or run:

source ~/.zshrc
