Install

Open your terminal app and paste these one at a time.

1
Download it

This downloads the meme-terminal folder (with all sounds) to your computer.

git clone <your-repo-url>
2
Install it

This copies the sounds into your home folder and tells your terminal to load meme-terminal every time it opens.

cd meme-terminal && ./install.sh
3
Restart your terminal

Close your terminal completely (Cmd+Q on Mac) and reopen it. Or run this:

source ~/.zshrc
✓
Test it

Hear a sound? You're done. Type garbage to hear the fahh.

meme-test fahh.mp3

Want custom sounds? Use the configurator above to generate your config, then paste it into ~/meme-terminal/meme.zsh
